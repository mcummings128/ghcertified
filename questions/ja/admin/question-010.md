---
question: "GitHub Enterprise Cloud で組織に対して SAML シングルサインオン (SSO) を有効にする際の主な影響は何ですか？"
documentation: "https://docs.github.com/en/enterprise-cloud@latest/authentication/authenticating-with-saml-single-sign-on/about-authentication-with-saml-single-sign-on"
---

- [x] SAML SSO により、組織のオーナーは組織リソースへのアクセスを制御および保護することができます。
- [ ] SAML SSO を有効にすると、IdP ユーザー全員のために新しい GitHub アカウントが自動的にプロビジョニングされます。
- [ ] SAML SSO が有効化されると、パブリックリポジトリにアクセスできなくなります。
- [ ] SAML SSO を有効にすると、メンバーは IdP ではなく GitHub を通じて認証する必要があります。
