- [x] ジョブレベル
- [x] ステップレベル
- [ ] ワークフローレベル
> `if:` はワークフローレベルでは使用できません。ワークフロー全体を条件付きでトリガーするには、アクティビティタイプを指定するなど、他の方法を使用する必要があります。
- [ ] 環境レベル
- [ ] 組織レベル
