- [x] Korzystając z webhooków, aby subskrybować informacje o zadaniach i uruchomieniach workflow, a potencjalnie także używając systemu archiwizacji danych.
- [x] Korzystając z metryk użycia i wydajności GitHub Actions dostępnych na poziomie organizacji i repozytorium.
> Metryki GitHub Actions są dostępne na karcie Insights na poziomie organizacji i repozytorium, zapewniając dane dotyczące użycia i wydajności, takie jak czasy uruchamiania, czasy oczekiwania i wskaźniki błędów.
- [ ] Licząc ręcznie uruchomienia workflow i zadania w każdym repozytorium na koniec każdego miesiąca.
- [ ] Użycie GitHub Actions nie może być śledzone; przedsiębiorstwa muszą szacować użycie na podstawie opinii programistów.
