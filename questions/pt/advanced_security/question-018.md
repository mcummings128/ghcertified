---
question: "Quando o GitHub realiza uma varredura de escaneamento de segredos em todo o código histórico nos repositórios da empresa, qual é o comportamento da notificação?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/managing-alerts-from-secret-scanning/monitoring-alerts#historical-scans"
---

- [x] O GitHub notifica os proprietários da empresa e gerentes de segurança, mesmo que nenhum segredo seja encontrado.
- [x] O GitHub notifica os administradores do repositório, gerentes de segurança e usuários com roles personalizados com acesso de leitura/escrita sempre que um segredo for detectado em um repositório.
- [ ] O GitHub notifica os proprietários da empresa e gerentes de segurança, somente se detectar segredos expostos.
- [ ] O GitHub notifica os autores dos commits que contêm segredos expostos.
