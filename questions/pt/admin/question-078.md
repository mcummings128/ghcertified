- [x] Usando webhooks para se inscrever em informações sobre trabalhos e execuções de workflow, e potencialmente utilizando um sistema de arquivamento de dados.
- [x] Usando as métricas de uso e desempenho do GitHub Actions disponíveis nos níveis de organização e repositório.
> As métricas do GitHub Actions são acessíveis na aba Insights nos níveis de organização e repositório, oferecendo dados de uso e desempenho como tempos de execução, tempos de fila e taxas de falha.
- [ ] Contando manualmente as execuções de workflows e trabalhos em cada repositório no final de cada mês.
- [ ] O uso do GitHub Actions não pode ser rastreado; as empresas devem estimar o uso com base no feedback dos desenvolvedores.
