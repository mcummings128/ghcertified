---
question: "Quais dos seguintes eventos podem acionar um workflow que não foi mesclado na branch padrão?"
documentation: "https://docs.github.com/en/actions/reference/workflows-and-actions/events-that-trigger-workflows#pull_request"
---

- [x] `push`
- [x] `pull_request`
> Para acionar um workflow via `pull_request`, a branch que contém o workflow deve ser a branch de destino do pull request.
- [ ] `repository_dispatch`
- [ ] `star`
- [ ] `issues`
- [ ] `issue_comment`
