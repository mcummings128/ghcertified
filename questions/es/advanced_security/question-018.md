---
question: "¿Cuál es el comportamiento de notificación cuando GitHub realiza un escaneo de secretos en todo el código histórico de los repositorios de la empresa?"
documentation: "https://docs.github.com/en/code-security/secret-scanning/managing-alerts-from-secret-scanning/monitoring-alerts#historical-scans"
---

- [x] GitHub notifica a los propietarios de la empresa y a los responsables de seguridad, incluso si no se encuentran secretos.
- [x] GitHub notifica a los administradores del Repository, a los responsables de seguridad y a los usuarios con roles personalizados con acceso de lectura/escritura siempre que se detecta un secreto en un Repository.
- [ ] GitHub notifica a los propietarios de la empresa y a los responsables de seguridad, solo si detecta secretos expuestos.
- [ ] GitHub notifica a los autores de los commits que contienen secretos expuestos.
